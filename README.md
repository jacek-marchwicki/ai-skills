# Personal AI Skills

This repository contains a collection of personal AI skills designed to extend the capabilities of AI agents using the `npx skills` tool.

## Installation

To use these skills, you first need to have the `npx skills` tool installed.

### Installing a Specific Skill

You can install a specific skill from this repository, for example, `download-screenshots`, using the following command:

```bash
npx skills add https://github.com/jacek-marchwicki/ai-skills --skill download-screenshots
```

This command adds the `download-screenshots` skill to your local skills environment, making it available for your AI agent.

## Skill Definition

Each skill in this repository is defined by a `SKILL.md` file. This file includes YAML frontmatter that specifies a unique name and a description for the skill, along with the instructions for the AI agent.
